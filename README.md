# collecters

## DEMO

  - デプロイしている場合はURLを記入（任意）

## 紹介と使い方

  - 前回の課題の続き
  - 趣味や好きなものをカテゴリーごとに登録できるアプリ
  - 登録した内容は他のメンバーと共有もできる

## 工夫した点

  - 画像を登録、削除できる機能を追加した
  - コメントの編集や削除機能も追加した

## 苦戦した点

  - 画像をデータベースに保存するところ（ほとんど参考サイトをもとに作ったので要復習）
  - 全員が見れる画面と、個々でログインした後のページがかなり複雑になってしまったので、フレームワークを使用すればよかった。
